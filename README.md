# sokoban_game
Sokoban (warehouse keeper) is a puzzle game where the player controls the warehouse keeper to place the boxes to the goal position.

Table Mapping of Each Square Content

Blank: White Space
Wall: '#'
Box: '$'
Keeper: '@'
Goal: '.'
Box + Goal: '*'
Keeper + Goal: '+'
